**`Project` :** To do list 

<br/>

## Quick Start

```
git clone https://github.com/heeyeon-lim/project-to-do-list.git

cd client
npm install
npm run start // run client

cd server
npm start // run server 
```

## 🌟 Features 

- 내용, 태그, 날짜를 포함하는 task 생성
- 하기 전, 진행 중, 완료 status를 기준으로 task 생성
- task 조회, 수정, 삭제
- task 검색
- 이름, 언어, 라이트/다크 모드 선택이 가능한 설정
